# gulp-pill
Gulp is a tool developed by NodeJS that helps you automate tasks such as compiling different languages, minifying code, auto-refreshing the browser when you modify your code, validating syntax, compressing files, creating custom automations and much more!

### Prerequisites

Use and understand gulp plugins.

### General Analysis

* Install gulp in your project
* Create a new Gulp Project
* Compile a script made in typescript
* Compile an SASS file to get its corresponding .css
* Apply CSS compression by minifying the content
* Create a task that makes use of "browser-sync"
* Create a task that makes use of the "Globbing" concept 
* Understand how .pipe () works in your tasks and what it is used for

### Deliverables

* Repository with code
* Create a correctly documented README file in the root directory of the project 
* Documentation of the project in PDF format
